Mes ordinateurs tournent sous [OpenSuse](https://www.opensuse.org/) Tumbleweed. Sur ceux-ci, j'utilise principalement [hyprland](https://hypr.land/) avec un thème *glassmorphism*. Mon terminal est un simple [kitty](https://sw.kovidgoyal.net/kitty/) avec [zsh](https://ohmyz.sh/). Mes serveurs sont sous [Debian](https://www.debian.org/index.fr.html) Trixie.

## Cross-Platform

### Firefox
[Firefox](https://www.mozilla.org/fr/firefox/new/) est navigateur sobre et efficace, basé sur Gecko. Il représente la seule alternative à Chromium et ses dérivés. J'utilise le thème [Gruvbox Dark](https://codeberg.org/calvinchd/GruvboxDarkFirefoxTheme), ainsi que plusieurs add-ons :

   - [µBlockOrigin](https://ublockorigin.com/fr) : bloqueur de pub léger et efficace propulsé par une communauté réactive, tant sur le dev que les listes de blocages.
   - [BitWarden](https://bitwarden.com/download/) : un gestionnaire de mot de passe qui se synchronise sur mon instance locale privée.
   - [Page-Assist](https://github.com/n4ze3m/page-assist) : me permet de dialoguer avec mon instance Ollama privée, directement au sein d'une page web.
   - [Ghostery](https://addons.mozilla.org/fr/firefox/addon/ghostery/) : Permet d'avoit l'auto-consent sur les bannières de cookies, en désactivant les choix optionnels par défaut, en plus d'éviter le tracking inter-sites.
   - [ClearURL](https://addons.mozilla.org/fr/firefox/addon/clearurls) : permet de supprimer automatiquement les trackers liés aux URL.

Il existe plusieurs navigateurs dérivés directement de Firefox : [LibreWolf](https://librewolf.net/), plutôt accès sur la vie privée, ou [ZenBrowser.](https://zen-browser.app/) plutôt accès sur la personnalisation et la productivité.

### Signal

[Signal](https://signal.org/fr/) est un système de messagerie instantanée entièrement chiffrée créé par les fondateurs de Whatsapp, après le rachat de ce dernier par Méta. 

### VSCodium
[VSCodium](https://vscodium.com/) est un fork, entièrement opensource, de Visual Studio Code, par Microsoft. Il est extensible et contient de nombreux thèmes. (pas sur mobile)

### OpenCode
[OpenCode](https://opencode.ai/) est une alternative à claude code prenant en charge une infinité de modèles, y compris github ou en local. (pas sur mobile)

## Android

### Obtainium
[Obtainium](https://obtainium.imranr.dev/) permet d'installer et de tenir à jour des apps depuis d'autres sources, comme github.<br />
[Github](https://github.com/ImranR98/Obtainium)

### Keyguard
KeyGuard est une application-compagnon pour bitwarden.</br>
[Github](https://github.com/AChep/keyguard-app) - [Play Store](https://play.google.com/store/apps/details?id=com.artemchep.keyguard) - [App Store](https://apps.apple.com/us/app/keyguard/id1402767401?l=fr-FR)

### Termix
[Termix](https://termix.site/) est client SSH moderne, européen et audité.
<br />[Github](https://github.com/Termix-SSH/Termix) - [Play Store](https://play.google.com/store/apps/details?id=com.karmaa.termix) - [App Store](https://apps.apple.com/be/app/termix-ssh-client-terminal/id6739386670)

### SpamBlocker
Comme son nom l'indique, permet de se débarasser d'appels entrants liés à des listes de spam.
<br />[Github](https://github.com/aj3423/SpamBlocker)


### Conduit
[Conduit](https://conduit.cogwheel.app/) est un client mobile natif pour openwebui, permettant d'utiliser ses propres LLM locaux comme assistants.
<br />[Github](https://github.com/cogwheel0/conduit) - [Play Store](https://play.google.com/store/apps/details?id=app.cogwheel.conduit&pli=1) - [AppStore](https://apps.apple.com/us/app/conduit-openwebui-client/id6749840287)

### Material Files
Material Files est une application qui permet de naviguer dans les dossiers android et distants facilement, et sans pubs.<br/>
[Github](https://github.com/zhanghai/MaterialFiles) - [Play Store](https://play.google.com/store/apps/details?id=me.zhanghai.android.files)

### ReVanced Manager
[RVM](https://github.com/vancedapps/rv-manager) permet d'installer des apk patchées, notamment pour Youtube ou Spotify.<br />
[Github](https://github.com/vancedapps/rv-manager)

## Linux

### btop
[btop](https://github.com/aristocratos/btop) est la version premios de `top` permettant de surveiller en temps réel l'allocation des ressources et la liste des processus.

### duf
[duf](https://github.com/muesli/duf) est la version premios de `du` permettant de surveiller l'espace disque.

### lsd
[lsd](https://github.com/lsd-rs/lsd) est la version premios de `ls` sous stéroïde.

## Windows

### WinGetUI
[WinGetUI](https://github.com/marticliment/UniGetUI) permet d'installer des paquets pour Windows depuis plusieurs stores / repos (Chocolatey, pip, scoop, winget, ...).

### FlowLauncher
[FlowLauncher](https://github.com/Flow-Launcher/Flow.Launcher) est un launcher et moteur de recherche qui remplace dans l'utilisation mon menu démarrer.

### Rufus
[Rufus](https://rufus.ie/fr/) est un utilitaire de création d'USB bootables. Il est rapide et prend en charge énormément de systèmes.

### Tabby
[Tabby](https://tabby.sh/) est un émulateur de terminal prenant en charge telnet, ssh, powershell, cmd, ...
